# Bolly Landing Page

A modern and responsive landing page developed as part of the **Toposel Frontend Development Assignment**.

## 📌 Project Overview

This project recreates the provided Bolly landing page reference using HTML, CSS, and JavaScript. The page includes a clean layout, responsive design, and interactive product animation.

## ✨ Features

- Responsive Landing Page
- Modern UI Design
- Interactive Navigation Bar
- 3D Mouse Hover Effect on Product Image
- Floating Product Animation
- Gradient Background Effects
- Hover Effects on Buttons
- Mobile Responsive Layout
- Clean HTML, CSS and JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Elementor (WordPress)
- LocalWP

## 🚀 How to Run

1. Download or clone this repository.
2. Keep all project files in the same folder.
3. Open `index.html` in any modern web browser.

No additional installation is required.

## 🎯 Assignment Objective

The objective of this assignment was to recreate the provided Bolly product landing page as closely as possible while maintaining responsive behavior and adding smooth user interactions.
